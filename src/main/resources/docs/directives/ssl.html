Directive can have multiple meanings. Each variant is separated with horizontal line

[edit] ssl

Syntax: ssl on | off
Default: off
Context: http
server
Reference: ssl


Enables HTTPS for a server. (Note that since nginx version 0.7.14, the standard way to enable SSL is through the listen directive.)


Module: HttpSslModule

[edit] ssl

syntax: ssl on | off

default: ssl off

context: mail, server

Enables SSL/TLS for this virtual server.


Module: MailSslModule
