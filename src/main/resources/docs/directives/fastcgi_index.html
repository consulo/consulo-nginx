[edit] fastcgi_index

Syntax: fastcgi_index name
Default:
Context: http
server
location
Reference: fastcgi_index


The name of the file which will be appended to the URI and stored in the variable $fastcgi_script_name if URI concludes with a slash.

For example: 

fastcgi_index  index.php;
fastcgi_param  SCRIPT_FILENAME  $document_root$fastcgi_script_name;

for the request "/page.php" parameter SCRIPT_FILENAME will be set to "/home/www/scripts/php/page.php", but for the "/" â€” "/home/www/scripts/php/index.php".
