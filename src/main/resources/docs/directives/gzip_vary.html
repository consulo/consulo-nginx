Directive can have multiple meanings. Each variant is separated with horizontal line

[edit] gzip_vary

Syntax: gzip_vary on | off
Default: off
Context: http
server
location
Reference: gzip_vary


Enables response header of "Vary: Accept-Encoding". Note that this header causes IE 4-6 not to cache the content due to a bug (see [1] ).
