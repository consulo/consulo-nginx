Directive can have multiple meanings. Each variant is separated with horizontal line

[edit] set

Syntax: set variable value
Default:
Context: server
location
if
Reference: set


Directive establishes value for the variable indicated. As the value it is possible to use a text, variables and their combination.

You can use set to define a new variable. Note that you can't set the value of a $http_xxx header variable.


Module: HttpRewriteModule

[edit] set

Assign a variable.

  • var — the variable.
  • value — its value. If it contains variable names, these will be evaluated.
